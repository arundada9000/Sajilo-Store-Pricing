Frequently asked questions

Everything you need to know about the product.

Do I need to pay for hosting?
No! Since this is a static template (HTML/CSS/JS), it can be hosted 100% free forever on platforms like GitHub Pages, Vercel, or Netlify. We provide a guide to help you set this up.
How does WhatsApp ordering work?
When a customer adds items to their cart and clicks checkout, the site automatically generates a formatted WhatsApp message with their order details and total sum. This message is sent directly to your WhatsApp number.
Do I need coding skills to manage products?
Products are managed in a simple JSON file or directly in the HTML depending on the template structure. It's copy-paste easy. If you don't want to touch any code, choose our Standard or Premium package and we'll do it for you!
Is it a one-time payment?
Yes, there are absolutely no monthly subscriptions or hidden fees. You pay once and own the source code forever.
